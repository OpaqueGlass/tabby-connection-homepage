# tabby-connection-homepage
**中文** | [English](README.md)

这是用于[Tabby](https://github.com/Eugeny/tabby)的SSH连接主页插件。

## 特点

- 读取配置文件中的SSH连接
- 提供标签页显示SSH连接主页,并配置编辑/克隆/删除
- 通过已有SSH配置快速连接
- 使用配置选择器或“配置和连接选项卡”作为启动时打开的主页
